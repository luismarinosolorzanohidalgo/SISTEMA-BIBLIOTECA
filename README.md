# 📚 SISTEMA-BIBLIOTECA

Sistema de gestión de biblioteca desarrollado en **PHP y MySQL**, diseñado para administrar de manera eficiente el catálogo de libros, usuarios y el control de préstamos y devoluciones. Permite organizar los recursos bibliográficos, mantener un registro detallado de los ejemplares y facilitar la consulta de información dentro de la biblioteca.

---

## 🛠 Funcionalidades principales

- ✅ Registro y administración de libros  
- ✅ Gestión de usuarios o lectores  
- ✅ Control de préstamos de libros  
- ✅ Registro de devoluciones  
- ✅ Consulta de disponibilidad de ejemplares  
- ✅ Organización del catálogo bibliográfico  

---

## 💻 Tecnologías utilizadas

- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript
